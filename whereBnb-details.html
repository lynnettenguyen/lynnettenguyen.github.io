Wherebnb

  • Frontend: React, Redux, HTML5, CSS3
  • Backend: Express, Node, Google Maps API, SQLite, Sequelize
  • Project Date: July 2022
  • Live Site: https://not-discord-app.herokuapp.com

Wherebnb is a web-application inspired by Airbnb. Wherebnb features Google Maps API, multi-page forms, and a responsive web design for a compelling user experience.
